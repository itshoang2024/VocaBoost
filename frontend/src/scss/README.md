# SCSS System

This folder contains the frontend styling system for VocaBoost.

## Entry Point

- `main.scss`: imports the token system and feature-level styles

## Core Layers

- `_DesignTokens.scss`: base design values
- `_ComponentTokens.scss`: component-level token composition
- `_CSSVariables.scss`: runtime CSS custom properties
- `_Fonts.scss`: font setup

## Feature Style Modules

- `Auth/`
- `User/`
- `Vocabulary/`
- `Classroom/`
- `Review/`
- `Admin/`
- `Components/`
